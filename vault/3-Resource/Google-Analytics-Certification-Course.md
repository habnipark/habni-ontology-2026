# Google Analytics Certification Course

## Summary
Google 공식 GA4 기초 인증 과정.

## Key Takeaways
1. GA4는 세션 기반이 아닌 이벤트 기반
2. 데이터 스트림이 기존의 뷰(View)를 대체
3. 탐색(Explorations)이 맞춤 보고서를 대체

## Notes
- 이벤트 모델: 모든 상호작용이 파라미터를 가진 이벤트로 추적됨
- [[GA4]] 속성 구조: Account > Property > Data Stream
- [[Session]]은 GA4에서 `session_start` 이벤트로 시작됨
